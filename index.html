Introduction （Scala3 Book 简介）

[目录] [下一篇: 2. Scala 3 Features （Scala3 特性） ]

欢迎阅读 Scala 3 手册。本书的目标是提供对 Scala 语言的非正式介绍。它以相对轻松的方式涉及所有 Scala 主题。如果您在阅读本书的任何时候想了解有关特定功能的更多信息，都可以找到我们的参考文档的链接 https://dotty.epfl.ch/docs/index.html ，其中更详细地介绍了 Scala 语言的许多新功能。

在本书的过程中，我们希望证明 Scala 是一种漂亮的、富有表现力的编程语言，具有干净、现代的语法，支持函数式编程 (FP) 和面向对象编程 (OOP)，并提供安全的静态类型系统。 Scala 的语法、语法和特性经过重新思考，在一个开放的过程中进行辩论，并在 2020 年更新，比以往任何时候都更清晰、更容易理解。

本书首先在“A Taste of Scala (初试 Scala) ”部分对Scala的许多特性进行了牛刀小试。导览结束后，接下来的部分将提供有关这些语言功能的更多详细信息
